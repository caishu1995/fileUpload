程序说明：
	做一个文件上传文件的控件，允许拖动、选择、替换等要求。
辅助说明：
    （用户）  1、用户定义对象，并定义一些辅助数据（√）
    （默认）  2、程序在用户定义的div中自动创建内容（√）
    （默认）  3、在js中创建的html代码（×）
    （默认）  4、支持选择和拖拽文件，保存文件信息，并展示部分信息（√）
    （默认）  5、支持替换文件（√）
    （默认）  6、支持交换文件（√）
    （默认）  7、支持删除文件（√）
    （默认）  8、在非成功状态中，支持双击名称时重命名（√）
    （默认）  9、在文件上传中，支持取消文件（√）
    （默认） 10、在文件失败后，支持重新上传文件（√）
    （默认） 11、支持统一提交文件（√）

文件说明：
    CSS ：样式
        baseStyle.css ：所有样式文件集合
        img ：图标集合
    JS  ：js文件
    	cutFile.JS    ：切分文件（暂不需要，仅用于文件过大，切分时候使用）
        fileHandle.js ：创建此控件的js集合

    CancelData.ashx ：取消的请求发送的位置
    DeleteData.ashx ：删除的请求发送的位置
    RecieveData.ashx：提交的请求发送的位置
    index.html      ：主页面

使用说明：
	在页面中创建一个div容器，然后模仿index.html的script内容，进行创建。
	包含css、js文件夹
	在页面中引用css、js文件夹。
